# InarkOS
InarkOS is a operating system(Virtual OS) written in 100% Python

#
# Installing

### 1. Clone the repository
    $ git clone https://github.com/InarkVOS/InarkOS
### 2. Open the directory
    $ cd InarkOS
### 3. Start the setup
    $ python3 bootscreen.py
